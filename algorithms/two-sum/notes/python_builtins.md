# Python 常用内置函数速查

> 记录我刷题时最常用的 Python 内置函数，方便随时查阅。

## 一、输入输出
- `print()`：输出到屏幕  
- `input()`：读入一行字符串

## 二、类型转换
- `int()`：转整数  
- `str()`：转字符串  
- `list()`：转列表  
- `set()`：转集合（自动去重）

## 三、数学运算
- `abs()`：绝对值  
- `max()` / `min()`：最大值/最小值  
- `sum()`：求和  
- `round()`：四舍五入

## 四、序列操作
- `len()`：长度  
- `sorted()`：排序（返回新列表）  
- `enumerate()`：同时取下标和值  
- `range()`：生成整数序列  
- `zip()`：打包多个序列

## 五、其他常用
- `type()`：查看数据类型  
- `isinstance()`：判断类型  
- `open()`：打开文件（了解即可）

## 使用建议
不需要死记硬背，写代码时用到了就来查，多用几次就自然记住了。
