## 二、算法与数据结构练习（计划中）
通过刷 LeetCode 或洛谷的基础题，提升逻辑能力。每道题都会记录思路、卡点、多种解法。

**计划练习的题目类型**（有代表性的基础题）：
- **数组**：两数之和、移动零、旋转数组
- **链表**：反转链表、合并两个有序链表
- **栈与队列**：有效的括号、用栈实现队列
- **树**：二叉树的中序遍历（递归与迭代）
- **动态规划**：爬楼梯、最大子数组和

我会把每道题的解题代码、思考过程、复杂度分析放在 `/algorithm` 文件夹下，并在 README 中更新链接。
